Quantum Mobile setup¶

Virtualbox VM¶

Ready to use AiiDA environment so-called Quantum Mobile is prepared by AiiDA team at https://materialscloud.org/work/quantum-mobile.

Recommended version is v21.05.1. https://quantum-mobile.readthedocs.io/en/latest/releases/versions/21.05.1.html

Older versions, v21.03.18-qe or v20.06.1 is OK except for pymatgen problem whose tentative solution is found at https://github.com/atztogo/aiida-tutorials/wiki.

Virtualbox¶

To use this, you need to install virtualbox on your computer. Virtualbox is downloaded at https://www.virtualbox.org/wiki/Downloads. It is recommended to install the one for your operating system (linux, too) from this web site.
